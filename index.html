WebPac - PacMan HTML5

Comandos

  1. ENTER -> inicia o jogo
  2. Q -> sai do jogo
  3. UP, DOWN, LEFT, RIGHT -> direciona o pacman
  4. -------------------------------------
  5. DEBUG
  6. A -> exibe área limite de colisão
  7. S -> exibe posições
  8. D -> exibe no console as coordenadas da área de colisão

Este site é melhor visualizado no Chrome, Firefox ou Safari
